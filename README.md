# Portfolio Website

Your personal portfolio website has been created!

## How to View
Simply open the `index.html` file in your preferred web browser (e.g., Chrome, Edge, Firefox).

## File Structure
- **index.html**: The main website file containing all your content.
- **styles.css**: Contains all the styling, colors, and responsive design rules.
- **script.js**: Handles the mobile menu toggle and scroll animations.

## Customization
- **Images**: You can add your own images (e.g., for a profile picture) by creating an `assets` folder and linking them in the HTML.
- **Colors**: You can change the color theme by editing the variables at the top of `styles.css`.
