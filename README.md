# Prajwal S Shetty | Portfolio Website

A premium, elegant, and minimal single-page personal portfolio designed for a professional AI & Cybersecurity Engineer. Built with a modern **Space Night** theme, glassmorphism cards, and interactive animations.

## Live Demo & Repository
- **Live Website**: [https://shetty7879.github.io/my-portfolio/](https://shetty7879.github.io/my-portfolio/)
- **GitHub Repository**: [https://github.com/Shetty7879/my-portfolio](https://github.com/Shetty7879/my-portfolio)

## Features
- **Space Night Theme**: Layered background gradients, animated ambient lights, and a subtle blueprint grid.
- **Glassmorphism Design**: Elegant floating glass navbar and cards with a frosted backdrop blur effect.
- **Micro-Animations**:
  - Typing animation script for hero headers.
  - 3D card mouse tilt effects.
  - Interactive infinite horizontal carousels with drag/swipe and snap alignment.
  - Spotlight cursor follower on desktop viewports.
- **Universal Lightbox**: Preloaded modal gallery for viewing academic certificates and awards.
- **SEO Optimized**: Complete metadata structure, Open Graph tags, sitemap.xml, and robots.txt.
- **Accessible & Responsive**: Fully keyboard navigable, semantic elements, and mobile-responsive overrides.

## Technologies Used
- **Structure**: Semantic HTML5
- **Styling**: Modern CSS3 (Variables, keyframe animations, grid layouts)
- **Logic**: Vanilla Javascript (DOM manipulation, IntersectionObserver, events)
- **Icons**: FontAwesome v6
- **Typography**: Google Fonts (Inter, Outfit, Space Grotesk, Patrick Hand)

## Folder Structure
```text
my-portfolio/
├── assets/               # Image resources, logos, and certificates
├── .gitignore            # Git exclusion rules
├── index.html            # Core HTML document
├── styles.css            # Stylesheet (Space Night theme)
├── script.js             # Interaction logic
├── sitemap.xml           # Search engine sitemap
├── robots.txt            # Search crawler directives
└── README.md             # Project documentation
```

## Installation & Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Shetty7879/my-portfolio.git
   ```
2. **Open the project**:
   - Simply double-click `index.html` to view the website in your browser.
   - Alternatively, serve it using a local development server like VS Code's **Live Server** extension.
